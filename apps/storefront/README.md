## Run (quick)

Install

Using pnpm (preferred):

```bash
pnpm install
```

Or using npm:

```bash
npm install
```

Dev server

```bash
pnpm dev
# or
npm run dev
```

Build

```bash
pnpm build
# or
npm run build
```

Tests

```bash
pnpm test
# or
npm run test
```

Note: If you encounter peer-dependency issues when installing test libs, try `npm install --legacy-peer-deps`.
